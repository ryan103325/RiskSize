# RiskSize｜台股現股當沖 風控倉位計算器

依「風險金額」自動算出可持有張數的當沖倉位計算器，已內含手續費與當沖證交稅，止盈價位也已扣除交易費用，確保設定的盈虧比是真實比例。

> 適用範圍：台股現股當沖。不含期貨、選擇權與零股；不計滑價、不檢查當沖額度與平盤下放空限制。

## 使用說明

1. 選擇交易方向：**做多** 或 **做空**
2. 輸入試算條件：
   | 欄位 | 說明 |
   |---|---|
   | 風險金額（千元） | 這筆交易願意承擔的最大虧損金額 |
   | 進場價 | 做多為買入價，做空為放空價 |
   | 止損價 | 觸價出場的價位 |
   | 盈虧比 | 用於反推止盈價位，預設 2 |
   | 手續費折數 | 券商手續費折扣，1～10 折，預設 6 折 |
3. 按「計算倉位」，即可看到可持有張數、部位所需資金、止盈價位、預期虧損 / 預期浮盈
4. 需要的話按「儲存這筆試算」留存紀錄，可於下方「歷史紀錄」查看、刪除或匯出 CSV
5. 展開「費率與預設值設定」可依券商折數或最新稅率調整計算基準

登入帳號後，設定與歷史紀錄可跨裝置同步；未登入則資料只存在目前使用的瀏覽器。

## 免責聲明

本結果未計入滑價與實際成交價落差，當沖急殺 / 急拉時實際虧損可能高於顯示值，僅供參考。
